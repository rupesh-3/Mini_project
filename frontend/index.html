How It Works

The NLP pipeline powering this summariser

🎯

Bahdanau Attention

Custom attention layer with Wa, Ua, Va weights lets the decoder focus on relevant encoder states.

BART Transformer

BLEU Scoring

Model accuracy evaluated using BLEU — the standard NLP metric for measuring generated text quality against references.
